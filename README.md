## Overview

WeddingUp is a SaaS platform designed to help wedding planners, couples, and vendors collaborate effectively throughout the wedding planning process.

### Key Features (Planned)

- **User Management**: Role-based accounts (Planner, Couple, Vendor)
- **Wedding Workspaces**: Project-based organization for each wedding
- **Task Management**: Kanban/List boards for tracking progress
- **File Uploads**: Centralized storage for wedding-related documents and images
- **Real-time Notifications**: Stay updated on changes and communications
- **Collaboration Tools**: Comments, chat, and shared timelines

## Tech Stack

- **Frontend**: Next.js, TailwindCSS, Shadcn/UI
- **Backend**: Supabase (Auth, Postgres, Storage, Realtime)
- **Hosting**: Vercel + Supabase
- **Testing**: Jest, Playwright
